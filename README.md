<h1 align="center">cc-hippo</h1>

<p align="center">A Claude-artifacts-style canvas app for UI design. Describe designs in chat, get artifacts on an infinite canvas.</p>

## Prerequisites

- [Claude CLI](https://docs.anthropic.com/en/docs/claude-code) installed and authenticated
- Node.js

## Quick Start

```bash
git clone https://github.com/meouwu-dev/cc-hippo.git
cd cc-hippo
npm install
npm run dev
```
